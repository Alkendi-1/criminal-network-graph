# Criminal Network Graph (CLI)

A Python command-line application for managing suspects and their accomplices using a graph-based structure.

## Overview
This application allows users to create and explore relationships between suspects by linking accomplices
and identifying indirect connections within a network.

## Features
- Add and manage suspects
- Link accomplices bidirectionally
- Display the full network
- Identify potential accomplices through indirect relationships
- Store data persistently using file serialization

## Technologies
- Python
- Graph-based data structures
- File handling and serialization

## About
This project was developed as part of academic studies to practice core programming concepts,
data structures, and logical problem-solving.
